PORTAL START KARNE KE LIYE

Backend:
1. Portal_Small_Files\backend folder open karein.
2. Dependencies install karein:
   pip install -r requirements.txt
3. Backend start karein:
   uvicorn main:app --reload

Frontend:
1. Portal_Small_Files\frontend folder open karein.
2. Dependencies install karein:
   npm install
3. Frontend start karein:
   npm run dev

Notes:
- node_modules, venv, database, uploads, .env aur large ortho_data copy nahi kiye gaye.
- Agar app ko .env chahiye ho, to frontend folder me naya .env banana padega.
- Backend default local URL: http://127.0.0.1:8000
- Frontend Vite local URL usually: http://localhost:5173
